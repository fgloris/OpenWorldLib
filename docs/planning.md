## SceneFlow Planning

### Code Optimization
- The current repository may still contain some redundant code, which we will further streamline;
- The interactions between different modules are not yet seamless enough. We will work towards enabling more flexible and natural integration between components;
- The performance of some existing methods still requires further improvement, specifically:
  - The sky layers generated in wonderworld are not yet satisfactory;

### 后续项目
- SceneFlow primarily aims to provide a more standardized framework for world models, along with corresponding inference code. Therefore, SceneFlow emphasizes comprehensiveness. Moving forward, we will develop training frameworks for world models based on several of the most mainstream foundational models;
- Alongside the release of the training framework, we will also provide corresponding datasets for users.
